# Чек-лист валидации устава КФА

Проверьте сгенерированный устав на соответствие требованиям законодательства КР и полноту информации.

## I. Структура документа

### Общая структура

- [ ] Документ имеет заголовок "УСТАВ"
- [ ] Указано полное наименование организации
- [ ] Присутствует информация об утверждении (протокол и дата)
- [ ] Указано место для регистрационных данных
- [ ] Присутствуют все 10 основных разделов (I-X)
- [ ] Статьи пронумерованы последовательно (1-36)
- [ ] Каждая статья имеет название
- [ ] Пункты внутри статей пронумерованы правильно (1.1, 1.2, и т.д.)

### Раздел I. Общие положения

- [ ] Указано полное наименование на русском языке (Статья 1.1)
- [ ] Указано сокращенное наименование (Статья 1.2)
- [ ] Указано наименование на кыргызском языке (Статья 1.3)
- [ ] Указано наименование на английском языке (Статья 1.4)
- [ ] Определена организационно-правовая форма (Статья 2.1)
- [ ] Перечислены все применимые законы КР (Статья 2.1)
- [ ] Указан статус: коммерческая или некоммерческая (Статья 2.2)
- [ ] Указан город местонахождения (Статья 3.1)
- [ ] Указан юридический адрес (Статья 3.2)
- [ ] Указаны почтовый и фактический адреса (Статьи 3.3, 3.4)
- [ ] Описан правовой статус юридического лица (Статья 4)

- [ ] **КРИТИЧНО**: Предусмотрено создание резервного фонда (Статья 17)
- [ ] **КРИТИЧНО**: Указан размер резервного фонда (Статья 17.2)
- [ ] **КРИТИЧНО**: Указаны источники формирования резервного фонда (Статья 17.3)
- [ ] **КРИТИЧНО**: Указаны цели использования резервного фонда (Статья 17.4)
- [ ] Минимальное количество членов не менее 3 человек (Статья 9.2)
- [ ] Определены паевые взносы членов (Статья 15)
- [ ] Указан размер паевого взноса (Статья 15.2)
- [ ] Определен порядок внесения паевых взносов (Статья 15.5-15.6)
- [ ] Устав может быть открыт для ознакомления (Статья 34.2)

### Требования к органам управления

- [ ] Общее собрание определено как высший орган (Статья 21.1)
- [ ] Определена периодичность очередных собраний (Статья 21.3)

- [ ] Создан коллегиальный орган управления между собраниями (Статья 22)
- [ ] Указан состав и срок полномочий коллегиального органа (Статьи 22.2-22.3)
- [ ] Определен исполнительный орган (Статья 23)
- [ ] Создана ревизионная комиссия или предусмотрен аудит (Статья 27)

### Специфика финансовых организаций

- [ ] Указано взаимодействие с Национальным банком КР (Статья 8)
- [ ] Упомянуты требования регуляторов финансового рынка (Статья 2.1)
- [ ] Предусмотрен внешний аудит (Статья 29)
- [ ] Указана специфика деятельности на рынке ценных бумаг (Статья 6)

### Статья 16. Вступительные взносы

- [ ] **ОБЯЗАТЕЛЬНО**: Перечислены источники формирования (минимум 2)
- [ ] **ОБЯЗАТЕЛЬНО**: Указаны цели использования (минимум 2)

### Статьи 18-19. Другие фонды и распределение доходов

## VI. Органы управления

### Статья 20. Структура

- [ ] Перечислены все органы управления (минимум 3)
- [ ] Общее собрание указано как высший орган
- [ ] Указан коллегиальный орган управления (Совет/Правление)
- [ ] Указан исполнительный орган

### Статья 21. Общее собрание

- [ ] Указана периодичность очередных собраний (минимум 1 раз в год)
- [ ] Определены основания для внеочередных собраний (минимум 2)
- [ ] Указан кворум (обычно 50%+1 или 2/3)
- [ ] Определено квалифицированное большинство (для важных решений)
- [ ] Определено простое большинство (для обычных решений)

### Статья 22. Совет/Правление

- [ ] Указано конкретное количество членов (рекомендуется 5-11)
- [ ] Определен срок полномочий (обычно 2-3 года)
- [ ] Указана частота заседаний (минимум раз в квартал)
- [ ] Определен кворум для заседаний
- [ ] Указан порядок принятия решений

### Статья 23. Исполнительный орган

- [ ] Указан тип органа (единоличный или коллегиальный)
- [ ] Определен порядок избрания/назначения
- [ ] Указан срок полномочий
- [ ] Определена подотчетность
