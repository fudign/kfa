# Performance Profiling Prime Prompt

Profile and optimize application performance.

## Usage

```bash
kfa prime use performance-profiling "Page load is slow"
```

## Prompt Template

Performance issue: {CONTEXT}

## Profiling Steps

1. **Measure Baseline**
   - Current metrics
   - Page load time
   - Time to interactive
   - Bundle size

2. **Identify Bottlenecks**
   - React Profiler
   - Chrome DevTools
   - Network waterfall
   - Database queries

3. **Analyze**
   - Rendering issues
   - Network issues
   - Database issues
   - JavaScript issues

4. **Optimize**
   - Code splitting
   - Lazy loading
   - Caching
   - Query optimization

5. **Measure Again**
   - Compare to baseline
   - Verify improvement
   - Check for regression

## Tools

- React DevTools Profiler
- Chrome Lighthouse
- WebPageTest
- Laravel Debugbar

## Expected Output

1. Performance report
2. Bottlenecks identified
3. Optimizations applied
4. Metrics improved

## Success Criteria

- Measurable improvement
- No regression
- Sustainable
