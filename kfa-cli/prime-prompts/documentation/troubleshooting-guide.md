# Prime Prompt: Troubleshooting Guide

Create troubleshooting guide for common issues and their solutions.

## Usage

```bash
kfa prime use documentation/troubleshooting-guide "Create DB connection troubleshooting guide"
```

## Prompt Template

I need a troubleshooting guide for:

**Context:** {CONTEXT}

### Guide Structure

1. **Common Issues**
   - List frequent problems
   - Group by category
   - Include error messages

2. **For Each Issue:**

   **Problem:**
   - Clear description
   - When it occurs
   - Error messages
   - Symptoms

   **Diagnosis:**
   - How to confirm it's this issue
   - Commands to run
   - Logs to check

   **Solution:**
   - Step-by-step fix
   - Commands to run
   - What to verify
   - Prevention tips

3. **Quick Reference**
   - Checklist of common fixes
   - Diagnostic commands
   - Quick solutions

4. **Advanced Debugging**
   - Enable debug mode
   - Logging configuration
   - Performance profiling
   - Where to find detailed logs

5. **Getting Help**
   - Information to include when asking for help
   - Log examples
   - System information
   - Reproduction steps

## Success Criteria
- ✅ Common issues covered
- ✅ Solutions tested
- ✅ Clear instructions
- ✅ Easy to navigate
