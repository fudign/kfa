# Onboarding Guide Prime Prompt

Create onboarding documentation for new developers.

## Usage

```bash
kfa prime use onboarding-guide "Create frontend developer onboarding"
```

## Prompt Template

Create onboarding guide for: {CONTEXT}

## Guide Structure

1. **Introduction**
   - Project overview
   - Tech stack
   - Architecture

2. **Setup**
   - Prerequisites
   - Installation steps
   - Configuration
   - Verify setup

3. **Codebase Tour**
   - Directory structure
   - Key files
   - Conventions
   - Patterns

4. **Development Workflow**
   - Creating branches
   - Running locally
   - Testing
   - Submitting PRs

5. **Common Tasks**
   - Add new feature
   - Fix a bug
   - Add tests
   - Deploy

6. **Resources**
   - Documentation links
   - Team contacts
   - Tools
   - Learning resources

7. **FAQs**
   - Common issues
   - Solutions
   - Tips

## Expected Output

Onboarding guide in docs/onboarding/

## Success Criteria

- New dev can start in 1 day
- Clear steps
- Up to date
- Tested with new hires
