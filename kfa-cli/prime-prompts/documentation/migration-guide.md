# Prime Prompt: Migration Guide

Create comprehensive migration guide for version updates or breaking changes.

## Usage

```bash
kfa prime use documentation/migration-guide "Create v2 to v3 migration guide"
```

## Prompt Template

I need a migration guide for:

**Context:** {CONTEXT}

### Guide Structure

1. **Overview**
   - What changed and why
   - Migration difficulty (easy/medium/hard)
   - Estimated migration time
   - Prerequisites

2. **Breaking Changes**
   - List all breaking changes
   - Impact of each change
   - Migration path for each
   - Code examples (before/after)

3. **Step-by-Step Migration**
   - Numbered steps
   - Commands to run
   - Code changes needed
   - Configuration updates
   - Testing steps

4. **New Features**
   - New capabilities
   - How to use them
   - Benefits over old approach

5. **Deprecations**
   - What's deprecated
   - Timeline for removal
   - Alternative approaches

6. **Troubleshooting**
   - Common issues
   - Solutions
   - FAQ

## Success Criteria

- ✅ Clear, actionable steps
- ✅ Code examples provided
- ✅ Common issues addressed
- ✅ Easy to follow
