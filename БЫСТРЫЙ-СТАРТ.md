## 🔧 Если что-то не работает:

### Ошибка: "Failed to fetch"

```bash
# Проверь что Laravel запущен:
cd kfa-backend/kfa-api
php artisan serve --port=8000
```

### Ошибка: "Unauthorized"

```
1. Выйди из системы (Logout)
2. Войди заново: admin@kfa.kg / password
3. Попробуй снова
```

### Ошибка: "CORS policy"

```bash
# Убедись что используешь порт 3001:
http://localhost:3001 (не 3000!)
```
